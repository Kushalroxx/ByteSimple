## To run the backend

### 1) Go to the server dir
```shell
cd server
```
### 2) Create a `.env` file
Follow the `.env.sample.txt` file and create one `.env` file
### 3) Install all packages
```
npm install
```
### 4) start the server
```
npm run start
```
## To run the Admin

### 1) Go to the admin dir
```
cd admin
```
### 2) Create the `.env` file 
Follow the `envExample.txt` file and create one `.env` file
### 3) Install all packages

### 4) Start the app
